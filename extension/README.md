# Rtrvr Microsoft Edge Extension

## Installation Instructions

### Step 1: Download the Extension
1. Click the "Download Extension" button on the extension page
2. Save the ZIP file to your computer

### Step 2: Extract the ZIP File
1. Right-click on the downloaded ZIP file
2. Select "Extract All" or use your preferred extraction tool
3. Choose a location to extract the files (e.g., Desktop or Documents)
4. Remember the location of the extracted folder

### Step 3: Open Microsoft Edge
1. Launch Microsoft Edge browser on your computer

### Step 4: Go to Extensions
1. Click the three dots (â‹¯) menu in the top-right corner of Edge
2. Hover over "Extensions" in the dropdown menu
3. Click "Manage extensions"

Alternatively, you can type `edge://extensions/` in the address bar and press Enter.

### Step 5: Enable Developer Mode
1. In the Extensions page, look for "Developer mode" toggle in the bottom-left corner
2. Toggle it ON (it should turn blue)

### Step 6: Load the Extension
1. Click the "Load unpacked" button that appears after enabling Developer mode
2. Navigate to the folder where you extracted the extension files
3. Select the extension folder (it should contain manifest.json, popup.html, etc.)
4. Click "Select Folder"

### Step 7: Start Using
1. The extension should now appear in your extensions list
2. You'll see the Rtrvr icon in your browser toolbar
3. Click the icon to open the AI agent interface
4. Start typing commands to interact with any webpage!

## Features

- **AI-Powered**: Uses advanced AI models to understand natural language commands
- **Works Everywhere**: Compatible with all websites
- **No Login Required**: Fully cloud-based, no account needed
- **Easy to Use**: Just type what you want to do in natural language

## Troubleshooting

If the extension doesn't work:
1. Make sure Developer mode is enabled
2. Check that all files are in the extension folder
3. Reload the extension by clicking the refresh icon
4. Check the browser console for any errors
